## git branch

**git branch** — показать список всех локальных веток в текущей директории.

`git branch`

**git branch -a** — посмотреть полный список локальных и удаленных веток.

`git branch -a`

**git branch -d [branch]** — удалить заданную ветку.

`git branch -d [branch]`
